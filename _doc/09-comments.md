### Comments

#### Multi-line comments

```typescript
/**
 * 😍 Cool 😍
 */

/**
 * 💡 This is a perfectly
 * 🍷 valid multi-line comment 😉
 */
```

```typescript
/**
 * 😱 Not cool 😱
 */

// this is a bad multi-line
// comment
```

#### Single-line comments

```typescript
/**
 * 😍 Cool 😍
 */

// 💡 This is a perfectly 🍷 valid single-line comment 😉
```

```typescript
/**
 * 😱 Not cool 😱
 */

/**
 * This is an invalid single-line comment
 */
```
