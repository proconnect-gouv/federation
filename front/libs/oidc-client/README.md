# Oidc-client

Provide method and components to achieve oidc connection.


## Component

```html
<RedirectToIdpFormComponent config={config} csrf={csrf} id={id}>
  <MyCustomButton>
</RedirectToIdpFormComponent>
```

Where `MyCustomButton` is a custom button to post the form.
